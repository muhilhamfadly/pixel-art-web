Tentang Pixel Art Converter

Pixel Art Converter merupakan sebuah sistem berbasis web yang dikembangkan untuk mengubah citra digital menjadi bentuk pixel art menggunakan teknik pemrosesan citra digital.

🎮 Apa itu Pixel Art?

Pixel art adalah bentuk seni digital di mana gambar direpresentasikan dalam unit piksel yang terlihat jelas. Gaya ini banyak digunakan pada game klasik era 8-bit dan 16-bit, serta saat ini kembali populer sebagai estetika visual retro.

⚙️ Cara Kerja

Pixel Art Converter menggunakan algoritma pixelation untuk mengubah gambar resolusi tinggi menjadi tampilan pixel art. Prosesnya:

  • Upload gambar dalam format JPEG, JPG, atau PNG
  • Resolusi gambar dikurangi menggunakan metode nearest neighbor untuk menghasilkan efek pixelation.
  • Warna citra disederhanakan menggunakan algoritma Median Cut untuk mengurangi jumlah warna.
  • Sistem menghasilkan beberapa variasi pixel art yang dapat dipilih dan diunduh oleh pengguna.

🛠️ Teknologi

Website ini dibangun dengan teknologi sederhana namun powerful:

  • Python dengan Flask sebagai backend framework
  • Pillow (PIL) untuk pemrosesan gambar
  • HTML5 dan CSS3 untuk antarmuka pengguna
  • Vanilla JavaScript untuk interaktivitas

✨ Fitur

  • Konversi cepat dan real-time
  • Variasi tingkat pixelation (8px – 48px)
  • Preview sebelum dan sesudah konversi
  • Download hasil dalam format PNG berkualitas tinggi
  • Drag and drop file upload
  • Responsive design untuk semua perangkat

🔒 Privasi

Gambar yang diunggah hanya digunakan untuk proses konversi sementara dan tidak disimpan secara permanen di server.
